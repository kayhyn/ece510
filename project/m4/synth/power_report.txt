Sequential storage and clock distribution dominate, consistent with the large
inferred per-lane weight register banks and full-array clock tree.

Final benchmark energy estimate
-------------------------------
The corrected production benchmark runtime is 42.714 ms/layer:

  1.018 W x 42.714 ms = 43.483 mJ/layer
  9.335 GFLOP/s / 1.018 W = 9.17 GFLOP/s/W

Limitations
-----------
* Detailed routing did not complete, so final wire parasitics and post-route
  buffer changes are absent.
* Default switching activity is used; the production tiled workload was not
  applied through SAIF/VCD power annotation.
* Power was analyzed under the 6.0 ns target constraint, while benchmark
  runtime uses the slower 114.536 MHz setup-limited projection. Multiplying
  1.018 W by that runtime is therefore a conservative, unscaled estimate.
* This is an estimate, not a measured wall-plug result.
